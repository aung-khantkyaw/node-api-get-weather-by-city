Weather by City API

Call our Express API to proxy Open-Meteo responses. Even when outbound network calls are blocked, we fall back to a curated mock so your demo never breaks.

Launch playground Open-Meteo docs

Quick Start

npm install
npm start
# or live reload
npm run dev

API Endpoints

  • GET /health
  • GET /v2/weather/{city}
  • GET /v2/weather?lat=...&lon=...
